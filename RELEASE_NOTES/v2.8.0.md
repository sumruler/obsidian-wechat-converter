## 🚀 多平台发布功能上线

Wechat Converter 现已支持**一键投递到知乎、掘金、CSDN、小红书、头条号等 20+ 内容平台**草稿箱，通过配套浏览器插件「Obsidian 发布助手」驱动。

**核心功能：**

- **在 Obsidian 写作，多平台分发**：在发布弹窗切换到「其他平台」标签，勾选目标平台，点击发布即可将文章投递到各平台草稿箱。微信公众号仍走原有 API 路径，两者互不影响。
- **浏览器登录态免登录**：浏览器插件使用你已在浏览器中登录的账号，无需在 Obsidian 内重复配置各平台账号。
- **连接安全**：通过配对令牌 + 握手机制确保 Obsidian 只与你自己安装的浏览器插件通信。
- **连接状态一目了然**：设置页实时显示已连接的浏览器实例（含浏览器图标）、连接时间和各平台上次登录状态。

**使用方式：**

1. 安装「Obsidian 发布助手」浏览器插件
2. 在 Obsidian 设置 → 其他平台 → 开启「启用浏览器插件发布」
3. 将浏览器插件中显示的连接令牌粘贴到设置页
4. 点击「测试连接」确认连通后，即可在发布弹窗中选择平台发布

### 🐛 问题修复

- 修复首次安装时设置页死锁（开关因令牌为空被禁用，但令牌输入需开启后才显示）
- 修复通过 Bridge 发布时本地图片双重编码导致的显示异常
- 修复封面图片为本地资源时无法传递给浏览器插件的问题
