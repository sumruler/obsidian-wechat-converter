title: 插件市场兼容性优化与平台清单修复
---

## 🔐 插件市场与兼容性优化

- 补充 README 英文摘要与隐私/权限说明，进一步明确插件的网络请求、本地文件读取、剪贴板写入、第三方账号与浏览器插件协作场景。
- 清理低兼容性 CSS 写法，减少 Obsidian 插件市场自动审核中的兼容性提示。
- 将发布平台禁用态样式从高级选择器改为显式状态类，提升旧版 WebView / Obsidian 环境下的样式稳定性。

## 🐛 问题修复

- 修复浏览器插件未连接时，设置页本地备用平台清单会错误显示 `WordPress`、`Typecho` 和 `Markdown 压缩包` 的问题。
- 这三个隐藏平台已从本地 fallback 平台清单中移除，因此断连、测试连接失败或首次加载时都不会再显示。

## 🧪 测试与发布验证

- 增加设置页平台选择器回归测试，确保隐藏平台不会从 fallback 清单暴露。
- 增加多平台发布弹窗禁用态样式测试，确保浏览器桥接未就绪时平台行正确标记为禁用。
- 已验证完整测试、构建和 release dry run 流程。
