---
title: 插件市场评分优化与本地 AI 模型支持
---

## ✨ 功能优化

- AI 编排请求超时默认值从 45 秒调整为 120 秒，更适合本地模型、慢速模型和较长文章的排版生成。
- AI Provider 的 Base URL 现在支持本机与局域网 HTTP 地址，方便连接 Ollama、LM Studio 或自建 OpenAI 兼容服务，例如 `http://localhost:11434/v1`。
- 添加 AI Provider 类型选择框的下拉箭头提示，让「类型」字段更明显地表现为可展开列表。

## 🔐 插件市场与安全透明度优化

- README 保持中文优先，同时补充英文摘要和隐私/权限说明，明确网络请求、本地文件读取、剪贴板写入、第三方账号和浏览器插件协作的使用场景。
- GitHub Release 资产调整为 Obsidian 官方推荐的三件套：`main.js`、`manifest.json`、`styles.css`，不再把 zip 与 `versions.json` 挂到 Release assets。
- Release workflow 增加 artifact attestation，提升发布产物来源透明度。
- 补充 `CONTRIBUTING.md`，说明开发、测试、Issue / PR 与兼容性原则。

## 🧰 构建与维护

- 移除 `builtin-modules` 开发依赖，改用 Node.js 内置模块列表。
- 新增 `tsconfig.json`，明确构建目标为 ES2018，减少自动审核误判。
- 更新 release 校验脚本，确保本地打包 zip 只包含官方三件套。
